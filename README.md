Import the numpy module to use the built-in functions for calculation

Prepare the lists from each linear equations and assign in np.array()

Using the np.linalg.solve(), we can find the solutions.
### Step 4: 
End the program
## Program:
```

import numpy as np


A = np.array([[1, 3], [2, 5]])
```
